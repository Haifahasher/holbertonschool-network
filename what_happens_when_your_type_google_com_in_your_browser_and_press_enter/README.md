# What Happens When You Type google.com in Your Browser and Press Enter?

This project contains a comprehensive blog post explaining the technical journey that occurs when you type `https://www.google.com` in your browser and press Enter.

## Overview

The blog post covers the complete flow of a web request, including:

1. **DNS Request** - How domain names are resolved to IP addresses
2. **TCP/IP** - The protocol suite that enables communication
3. **Firewall** - Security checkpoints protecting the network
4. **HTTPS/SSL** - Secure communication protocols
5. **Load-balancer** - Traffic distribution mechanisms
6. **Web server** - HTTP request handling
7. **Application server** - Business logic processing
8. **Database** - Data storage and retrieval

## Files

- `0-blog_post`: The complete blog post article covering all aspects of the web request flow

## Learning Objectives

By reading this blog post, you will understand:

- How DNS resolution works
- The TCP/IP handshake process
- The role of firewalls in network security
- How SSL/TLS establishes secure connections
- How load balancers distribute traffic
- The difference between web servers and application servers
- How databases interact with application servers
- The complete request-response cycle
