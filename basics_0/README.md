# Network Basics 0

This directory contains exercises and scripts covering fundamental networking concepts.

## Files

- **0-OSI_model**: Multiple choice questions about the OSI (Open Systems Interconnection) model
- **1-types_of_network**: Questions about LAN, WAN, and Internet network types
- **2-MAC_and_IP_address**: Questions about MAC addresses and IP addresses
- **3-UDP_and_TCP**: Questions about UDP and TCP protocols
- **4-TCP_and_UDP_ports**: Bash script that displays listening TCP and UDP ports with PID and program name
- **5-is_the_host_on_the_network**: Bash script that pings an IP address 5 times to check if a host is on the network

## Topics Covered

- OSI Model layers and organization
- Network types (LAN, WAN, Internet)
- MAC addresses and IP addresses
- TCP vs UDP protocols
- Network ports and sockets
- ICMP and ping utility
